# fastapi-demo

## Prerequisites

Install [pipenv](https://pipenv.pypa.io/en/latest/).

If you install to your user home (i.e. --user), you might have to add `PATH=~/AppData/Roaming/Python/Python39/Scripts:$PATH` to 
your .bash_profile to add `pipenv` to your path.

## Running the Application

```
uvicorn app:app --reload
```

## Testing with Swagger
